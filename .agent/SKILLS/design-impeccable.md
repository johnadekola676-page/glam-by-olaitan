# Skill: Impeccable (Design Vocabulary & Polish)

## Commands
- **`polish`**: When this command is run, the agent must audit the entire UI for:
    - **Optical Alignment:** Ensuring icons are visually centered, not just mathematically.
    - **Consistent Rhythm:** Spacing must be in multiples of 4 (4px, 8px, 16px, 32px).
    - **Visual Hierarchy:** Is the H1 clearly more important than the H2? Is the CTA the most vibrant element?
    - **Contrast Ratios:** Ensure text is readable against the Rose Gold backgrounds.

## Design Vocabulary
- **Tracking:** Tighten large headings `letter-spacing: -0.02em`.
- **Leading:** Increase body text line-height to `1.6` or `1.7` for luxury readability.
- **Elevation:** Use layered shadows instead of single dark blurs.
    - `box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1);`
