# Skill: Taste (High-End Aesthetics)

## Aesthetic Rules
- **No Generic Gradients:** Avoid `linear-gradient(to right, red, blue)`. Use mesh gradients or subtle multi-stop overlays in the same color family (e.g., Deep Rose to Warm Gold).
- **Typography as Art:** Move beyond 'Inter'. Use high-contrast pairings:
    - *Display:* 'Cormorant Garamond' (Serif)
    - *Body:* 'Jost' or 'Satoshi' (Modern Geometric)
- **Whitespace is Luxury:** If a section feels crowded, double the padding. Luxury is the ability to "waste" space.
- **Subtle Shimmer:** Use low-opacity animated masks to give a "metallic" feel to gold accents.

## The "Anti-AI" Protocol
- Never use the exact same border-radius on everything. 
- Use `rounded-2xl` for cards but `rounded-full` for buttons. 
- Avoid pure black `#000`. Use a very deep brown-gray `#1A1614` for text.
